# Project Skills

This directory is for repository-local skills.

Use one folder per skill:

```text
.agents/
  skills/
    my-skill/
      SKILL.md
```

Good candidates for local skills in this project:

- `langgraph-step-builder`: add the next learning step safely
- `graph-debugger`: inspect state transitions and routing
- `tool-loop-lab`: experiment with tool calling loops
- `memory-checkpointer`: add and test persistence

Keep each skill focused on one repeatable workflow.
